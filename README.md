Customer Churn Analysis System
📌 Overview

The Customer Churn Analysis System is a data analysis project that focuses on identifying customer behavior and understanding the factors that lead to customer churn.

Churn refers to customers leaving a service or stopping the use of a product. This project uses synthetic data and Exploratory Data Analysis (EDA) to detect patterns, trends, and key drivers of churn.

🎯 Objectives
Generate synthetic customer dataset
Analyze customer behavior
Identify high churn and low churn segments
Understand factors affecting churn
Perform group-based and relationship analysis
Provide business insights to reduce churn
📂 Dataset Description

The dataset is synthetically generated with 100,000 customer records.

Column Name	Description
Customer_ID	Unique ID
Age	18–70
Gender	Male / Female
Tenure	Years with company (0–10)
Balance	Account balance (0–200,000)
CreditScore	300–900
EstimatedSalary	10,000–150,000
NumOfProducts	1–4
IsActiveMember	Yes / No
Churn	0 = No, 1 = Yes
